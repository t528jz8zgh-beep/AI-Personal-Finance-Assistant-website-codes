Brand Introduction

A smart and warm finance dashboard built for everyday money clarity.

AIPF is a personal finance dashboard designed to make expense tracking feel clear, gentle, and visually engaging. It combines daily record keeping, budget monitoring, goal management, text-based expense parsing, charts, and word cloud insights in one connected experience. The goal of the brand is to turn financial data into something easier to understand, easier to manage, and more motivating to return to every day.

Project Description

Understand the dashboard logic, learning purpose, and text-mining workflow at a glance.

🧠 Project Logic & Course Relevance

LLM in Finance is the background of this project. We used AI tools to build a simple HTML webpage focused on finance-related daily use, mainly including expense tracking, basic data statistics, chart generation, and a spending word cloud.
Core Functions

The functional scope is intentionally lightweight, but the page still covers a complete mini workflow from recording expenses to summarizing spending patterns through charts and keyword visualization.

AI Collaboration

Beyond the functions themselves, we also used mainstream AI tools to optimize the page structure, bilingual version, and visual presentation, so the final result is clearer, cleaner, and more suitable for demonstration.
